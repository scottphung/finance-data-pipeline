# 📊 Quant Hedge Fund System (End-to-End ML + Portfolio Simulation)

A full-stack quantitative finance system simulating hedge fund workflows:
data ingestion → feature engineering → portfolio optimization → risk metrics → backtesting → interactive dashboard.

---

## 🚀 Project Overview

This project builds a simplified hedge fund analytics pipeline using:

- FastAPI (backend API)
- Streamlit (frontend dashboard)
- Pandas / NumPy (data processing)
- Portfolio optimization (mean-variance model)
- Backtesting engine (equity curve simulation)

The system allows users to:
- Select stock tickers
- Generate optimized portfolio weights
- Compute risk metrics (Sharpe Ratio, VaR)
- Visualize portfolio equity curve over time
