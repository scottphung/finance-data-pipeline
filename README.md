# Financial Data Pipeline

## Overview
This project collects stock market data using Python and stores it for analysis.

## Features
- Automated stock data collection
- Multi-stock support (AAPL, TSLA, MSFT)
- CSV data storage pipeline

## Tech Stack
- Python
- Pandas
- yfinance
- Git/GitHub

## Goal
Build an end-to-end financial data pipeline using real market data.
